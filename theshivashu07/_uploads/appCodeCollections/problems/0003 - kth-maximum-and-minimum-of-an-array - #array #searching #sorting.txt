Given an array arr[] and an integer K where K is smaller than size of array, the task is to find the Kth smallest and biggest element in the given array. It is given that all array elements are distinct.
Note :-  left and right denotes the starting and ending index of the array.

Example 1:
Input:  N = 6,  arr[] = 7 10 4 3 20 15,  K = 3
Output : 7 10

Example 2:
Input:  N = 5,  arr[] = 7 10 4 20 15,  K = 4
Output : 15 7
